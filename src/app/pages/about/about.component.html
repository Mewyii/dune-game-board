About

This is a home made web project for the dune imperium game from direwolf digital.

It contains a card configurator to create new imperium cards for the game, a leader configurator to create new leaders and a fully playable custom game board including ai. The only things that are not digitalized yet are the imperium cards, intrigues and tech tiles, which need to be used from the board game.



You can also find this project on github in case you heave any ideas for improvement or find any bugs.

https://github.com/Mewyii/dune-game-board
